# Smart IP, DHCP & Security Analyzer

Single-file browser analyzer for Cisco-style IP, DHCP, ARP, MAC table, and security evidence.

## Run

```bash
npm install
npm start
```

Open http://localhost:4173/.

## Test

```bash
npm test
```

The smoke test loads `index.html`, runs the built-in self-test, clicks the sample data button, and verifies that the analysis renders.
